# Clone Website

A simple, clean static website served with Express.js.

## Tech Stack
- **Runtime**: Node.js
- **Server**: Express.js
- **Frontend**: HTML, CSS (static files in `/public`)

## Project Structure
```
├── server.js          # Express server (serves static files on port 5000)
├── package.json       # Node.js dependencies
├── public/
│   ├── index.html     # Main HTML page
│   └── style.css      # Stylesheet
└── replit.md          # This file
```

## Running
```bash
npm install
npm run dev
```

The server runs on port 5000 (0.0.0.0).
